App Overview

The laptop inventory system is a prototype web application built to replace Excel-based tracking used by the IT team. It centralizes laptop data into structured forms and views, allowing records originally kept in spreadsheets to be imported and managed in a dashboard-style interface. The prototype uses Node.js + Express, MongoDB, and EJS, focusing on data structure and CRUD workflows.

Features Of The App

  • Imports laptop records from Excel into the structured database.
  • Displays laptops in a searchable inventory dashboard.
  • Can add, edit, update, and return laptops (CRUD)

Challenges

  • Converting loosely formatted Excel data into structured database fields.
  • Designing a UI that works smoothly for IT admin usage.
