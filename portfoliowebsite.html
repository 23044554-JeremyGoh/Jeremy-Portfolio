Portfolio Website Design

Design Purpose

To design a portfolio website with 4 tabs: Home, Portfolio, About, Contact & have at least 6 pieces of work to display on my portfolio page for my graded assignment

Design Rationale

My goal is to create a cohesive retro-inspired experience that balances aesthetics with functionality.

The homepage establishes the theme through "computer window" designs, pixelated fonts, and grain textures, setting a nostalgic tone.

The portfolio page adapts this layout into a structured grid for showcasing work, complemented by a consistent "VIEW MORE" button.

The about page highlights skills using traced icons paired with text for clarity and visual appeal.

Finally, the contact page blends retro aesthetics with usability through rounded rectangles and clear input placeholders. This unified design reflects my style and ensures an engaging user experience.
