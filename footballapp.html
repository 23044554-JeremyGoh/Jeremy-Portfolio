Football Apparel App

App Overview

The football apparel platform is an e-commerce application designed to showcase and manage a catalog of football-related products. Built with Node.js and Express, the backend integrates a MySQL database for storing product information and uses the EJS templating engine for rendering dynamic pages. It supports product CRUD (Create, Read, Update, Delete) operations and includes features for handling image uploads.

Features Of The App

  • Displays all available football apparel in a user-friendly interface.
  • Provides detailed information about individual products, including name, description, price, and image.
  • Admins can add new items to the inventory, including optional image uploads.
  • Existing product details, including images, can be updated via an intuitive form.
  • Admins can remove products, keeping the catalog updated.

Challenges

  • Maintaining a smooth user experience for both end-users browsing products and admins managing the inventory.
  • Ensuring the backend can handle a growing catalog of products and increasing user traffic.
  • Handling image uploads and updates efficiently, ensuring file system consistency.
