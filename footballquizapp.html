Football Quiz App

App Overview

The Football Club Quiz App is an interactive mobile application designed to test users' knowledge of football clubs through a fun and engaging quiz. Users are presented with images of football club logos and tasked with identifying the correct club name from a dropdown list of options. The app provides immediate feedback by calculating the user's score upon submission, encouraging learning and entertainment for football enthusiasts.

Features Of The App

  • Tracks user answers in real time.
  • Evaluates correctness upon submission and displays the score using an alert.
  • Ensures smooth scrolling for better user experience on devices with smaller screens.

Challenges

  • Keeping track of multiple answers and updating them dynamically as users made selections.
  • Dynamically loading local assets (club logos) and ensuring compatibility across different screen resolutions.
